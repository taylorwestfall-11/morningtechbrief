# Morning Tech Brief

Daily personalized tech news brief, generated by a scheduled Claude remote agent.

## How it works

1. A scheduled remote agent (cron `0 12 * * *`, i.e. 5am Pacific daily) clones this repo.
2. The agent reads `brief_prompt.txt` for instructions and `morning_brief.html` for the layout/styling template.
3. It runs web searches across the configured tech sources for stories from the last 24 hours.
4. It regenerates `morning_brief.html` with today's stories, appends to `tech_news_archive.md`, and pushes the changes back to this repo.

## Files

- `brief_prompt.txt` — instructions the daily run reads in (sources, profile, formatting rules).
- `morning_brief.html` — the live brief. Replaced fresh every morning.
- `tech_news_archive.md` — running archive of top stories by date (created on first run).

Enable GitHub Pages on this repo (Settings → Pages → deploy from `main`) and bookmark
